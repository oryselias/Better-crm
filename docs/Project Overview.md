# Project Overview

## Better CRM - Health CRM & LIS

An AI-powered healthcare CRM and Laboratory Information System for clinics.

## Mission

Build a premium, modern admin product that can:
- Authenticate staff with Supabase
- Enforce clinic-scoped data access with RLS
- Manage patients
- Manage appointments
- Ingest lab reports into a review queue
- Capture append-only audit events

## Tech Stack

- **Framework**: Next.js (App Router, React 19)
- **Styling**: Tailwind CSS v4
- **Database**: Supabase (Postgres with RLS)
- **Integrations**: Twilio/WhatsApp API
